# Next Day Movers — website

Marketing site for Next Day Movers — fully insured house, flat, office and single-item
removals and licensed waste clearance across London & Kent.

Static site: plain HTML/CSS/JS, no build step. Just deploy the folder.

## Pages
- `index.html` — home
- `services.html` — services
- `gallery.html` — portfolio gallery (lightbox)
- `quote.html` — instant quote
- `contact.html` — contact + message form
- `removals-*.html` — location landing pages (Bromley, Croydon, Dartford, Sevenoaks, Maidstone, Bexley)
- `404.html`, `sitemap.xml`, `robots.txt`

## Assets
- `assets/site.css`, `assets/site.js` — shared styles + behaviour
- `assets/photos/` — job photos (web-optimised)
- `assets/logo.svg`, `assets/favicon.*`, `assets/og-image.png`

## Forms
Quote + contact forms email **nextdaymoversuk@gmail.com** via FormSubmit (no backend).
